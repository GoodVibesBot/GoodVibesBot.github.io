About

With many features like good vibes, sniping, and emojis why wouldn't you have this bot? There is still way more to come, with new features and just updates for more user ease.

Commands
Help                         

%help

This message will show an embed with most of the commands to show, as well as the message to execute the commands.

Vibe

%vibe

Some selected responses are given out at random. The responses include phrases to lighten up your mood and give you some good vibes.

Secret Help

%secrethelp

This message will show an embed of all of the emojis that are from different severs that you can use without the need for nitro, as well as the message to execute the commands.

Wholesome

%wholesome

This command will give selected Gifs to provide wholesome vibes with powerful phrases.

Snipe                         

%snipe

This command will snipe or retrieve a users last deleted message from a certain text channel. This command does have a time-limit of 1 minute before the users message is deleted.

Editsnipe

%editsnipe

This command will snipe or retrieve a users last edited message from a certain text channel. This command does have a time-limit of 1 minute before the users edited message is deleted.

Pets

%pets

This command will give pictures of pets from my friends. Most pictures are taken from the #pets channel and put into my bot. Keep in mind, there are a lot of pictures, it can be very easy or hard to find an exact image of a pet because images are selected randomly.

Breathe                   

%breathe

This command will give a gif that has a simple breathing exercise to calm down to.
